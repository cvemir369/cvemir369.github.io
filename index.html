Chordpad Android App

Chordpad is an Android app made with Python and Kivy library.

"Kivy is an Open source Python library for rapid development of applications
that make use of innovative user interfaces, such as multi-touch apps."

"Chordpad" is a music related text editor with an easy chord and
song structure input keyboard for easier song chords notation.
It saves created pads in bundled with Python file-based SQL database - SQLite3.
App includes multiple screens and full CRUD functionality.
